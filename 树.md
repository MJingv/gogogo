## 二叉树

### 基本概念
- 分类：满二叉树、完全二叉树、非完全二叉树
- 存储结构
  - 数组（适合满/完全二叉树）
    - 父 n/2
    - 左 2n
    - 右  2n+1
  - 链表（最常用）   

### 递归遍历 O(n)
- 前序（根左右）
- 中序（左根右）
- 后序（左右根）

```js
const preOrder=(root)=>{
    if(root!==null) {
        console.log(root.val)
        preOrder(root.left)
        preOrder(root.right)
    }
}
```

- 中序非递归遍历
```js

let list =[] //空栈

        list.unshift(p) //根入栈

     }else{
        let q=list.shift() //左孩子出栈
        console.log(q.val) //打印根节点

             }
        }
    
}
```
