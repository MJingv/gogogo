# css动画为什么比js高效？分层与合成

如何渲染一帧：合成、重排、重绘

浏览器如何展示图像：

- 显示器1s读60次前缓冲区的图像展示到显示器上
- 显卡合成新图像放在后缓存区
- 前缓存区与后缓冲区交换展示图像

浏览器如何实现合成：
分层：将素材分成多个图层
合成：以上图层合并。合成操作在合成线程完成，不会影响主线程。
分块：在首次合成图块时使用分辨率较低的图片，之后替换。

css比js动画块

- 渲染引擎将willchange/transform/opacity属性的图层单独图层，不会涉及到主线程
- 单独图层占内存，合理使用
